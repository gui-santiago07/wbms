# Option7 - Sistema de Monitoramento de Produção

Sistema de monitoramento OEE (Overall Equipment Effectiveness) para chão de fábrica com métricas em tempo real, controles de máquina e análise de performance.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
